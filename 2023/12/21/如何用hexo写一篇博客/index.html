如何用hexo写一篇博客

发表于2023-12-21|更新于2023-12-28|hexo
|字数总计:413|阅读时长:1分钟|阅读量:

1. 创建文件

打开终端输入命令：cd ~/myblog
请将myblog替换成你的博客主文件夹的实际路径，如下图：

在出现password后输入密码，出现如下图所示即为创建成功

2. 更改权限

创建成功后可以在主文件夹的source/_posts中找到新建markdown文档

右键该文档，点击“显示简介”

点击右下角小锁图标，使用Touch ID或者密码解锁，图标变成开锁后在下方staff栏右侧“只读”处更改为“读与写”

共享与权限下方出现“你可以读和写”即表示权限更改成功

3. 博客编辑

打开刚才生成的markdown文档，默认为如下界面

我们可以随便编辑点内容
可以在tags后添加如下格式的内容：-tags（将tags换成你想要的tag）
可以添加categories，格式如下：categories：category（将category换成自己的内容）

4. 保存&发布

按下command+s保存markdown文件，进入终端，依次执行
sudo hexo g
sudo hexo s
第一条命令用于文件保存以及页面生成，第二条用于本地服务器运行，若有以下输出则表示成功

打开终端输出的端口，通常为 http://localhost:4000/
检查博客是否载入

去Categories页面和Tags页面检查是否载入

如上则完成，后续可在终端界面按下control+c关闭服务运行，然后输出命令进行git的同步
sudo hexo d

出现Deploy done即为成功

经过如上步骤，你已经可以完成一篇博客的发布了，快去试试吧
